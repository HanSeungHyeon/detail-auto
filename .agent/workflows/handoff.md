---
description: 퇴근 시 작업 내용을 정리하고 Git에 푸시하여 인수인계를 준비합니다.
---

이 워크플로우는 사용자가 작업을 마치고 다른 장소(집/회사)에서 이어하기 위해 "퇴근" 또는 "작업 종료"를 요청할 때 수행합니다. 
cmd 명령어 시 cmd /c를 같이 작성할것

// turbo-all
1. 현재 작업 상태를 요약하여 인수인계 문서를 업데이트합니다.
   `npm run handoff`

2. 변경된 모든 파일을 스테이징합니다.
   `git add .`

3. 인수인계 메시지와 함께 커밋을 생성합니다.
   `git commit -m "Handoff: 업무 장소 이동을 위한 자동 커밋"`

4. 원격 저장소에 푸시합니다.
   `git push`

5. 사용자에게 푸시 완료 및 다음 장소에서의 가이드를 안내합니다.
